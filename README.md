# RFM-Customer-Segmentation-Analysis-Python-SQL-Pandas-NumPy-Matplotlib
Customer Segmentation using RFM Analysis

This project applies RFM (Recency, Frequency, Monetary) Analysis to segment customers based on their purchasing behavior using the Online Retail dataset. The objective is to identify high-value customers, understand customer engagement patterns, and support data-driven marketing strategies.

The project begins with comprehensive data cleaning and preprocessing, including handling missing values, removing duplicates, filtering canceled transactions, and converting date columns into appropriate formats. Key RFM metrics were then calculated:

1. Recency: Days since the customer's last purchase.
2. Frequency: Total number of transactions made by the customer.
3. Monetary: Total revenue generated by the customer.

Customers were scored and categorized into meaningful segments such as Champions, Loyal Customers, Potential Loyalists, At-Risk Customers, and Lost Customers. SQL queries were used for customer-level analysis and revenue insights, while Python libraries such as Pandas, NumPy, Matplotlib, and Seaborn were used for data manipulation and visualization.

Key Insights
Identified the most valuable customer segments driving revenue.
Analyzed customer distribution across different RFM groups.
Measured segment-wise revenue contribution.
Highlighted at-risk customers for retention campaigns.
Provided actionable recommendations for targeted marketing and customer engagement.
